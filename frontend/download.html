Get Fabric API from Modrinth

MonoLit needs Fabric API to load. We don't host it ourselves — grab the official jar from Modrinth in 30 seconds.

  1. 1

    Open the Fabric API page on Modrinth

    Click here: modrinth.com/mod/fabric-api ↗

  2. 2

    Filter by Minecraft version

    Click the "Versions" tab. In the filter bar, set Game versions → 1.21.11 and Loaders → Fabric.

  3. 3

    Download the latest matching version

    Hit the green Download button on the topmost row. The file looks like fabric-api-0.XXX.X+1.21.11.jar.

  4. 4

    Drop it in your mods/ folder

    The animated guide below shows exactly where the folder lives on Windows / Mac / Linux.
