Install in 90 seconds

From clean Vanilla to MonoLit-ready. Five steps, two jars.

1

Install the Fabric loader

Run the official Fabric Installer for 1.21.x. Pick "Client" and let it generate the launcher profile.

2

Drop in Fabric API

Grab Fabric API from Modrinth (matching 1.21–1.21.11) and place the jar in your %appdata%/.minecraft/mods/ folder.

3

Drop in MonoLit Lite

Download MonoLit Lite above and put the jar next to Fabric API in mods/. That's it — no installer.

4

Launch the Fabric profile

Open the Minecraft launcher → choose the fabric-loader-1.21.x installation → Play. The MonoLit logo should appear on the title screen.

5

Open the build menu

In-game press K to open the schematic browser, or hold Ctrl while placing a block to start a Bedrock-style line.

FAQ

Is MonoLit allowed on DonutSMP?

Yes. MonoLit's auto-builder uses adaptive packet pacing that stays under DonutSMP's anti-cheat thresholds. Use Lite freely; we test against Donut every release.

Why do I need Fabric API?

Fabric API exposes the modding hooks (registries, networking, events) that MonoLit relies on. It's a single jar — drop it next to MonoLit in your mods/ folder.

Will MonoLit work with OptiFine / Sodium?

Sodium yes — fully compatible. OptiFine is Forge-only; if you need shaders, run Sodium + Iris instead.

Can I import my own schematics?

Lite supports up to 1 MB .schem imports. Pro removes the size cap and adds .litematic support.
