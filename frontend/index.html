Core Features

Built-in Schematics

Auto-Builder

FAQ

Is MonoLit allowed on DonutSMP?

Yes. MonoLit's auto-builder uses adaptive packet pacing that stays under DonutSMP's anti-cheat thresholds. Use it freely; we test against Donut every release.

Why do I need Fabric API?

Fabric API exposes the modding hooks (registries, networking, events) that MonoLit relies on. It's a single jar — drop it next to MonoLit in your mods/ folder.

Will MonoLit work with OptiFine / Sodium?

Sodium yes — fully compatible. OptiFine is Forge-only; if you need shaders, run Sodium + Iris instead.

Can I import my own schematics?

MonoLit supports up to 1 MB .schem imports. Larger files and .litematic support are on the roadmap.
